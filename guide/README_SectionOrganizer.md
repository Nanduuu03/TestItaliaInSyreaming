# 🧩 SectionOrganizer – Guida

Questa guida spiega come creare e riordinare le **sezioni personalizzate dei vari plugin** utilizzando SectionOrganizer.

Con questo plugin puoi:
- scegliere quali sezioni dei plugin mostrare
- riordinarle come preferisci
- creare una **Homepage personalizzata**

---

### 1. Apri le impostazioni del plugin

---

### 2. Configura le sezioni delle estensioni
1. Vai su **Configura sezioni estensioni**
2. Seleziona le **sezioni dei vari plugin** che vuoi aggiungere alla homepage
3. Premi l'icona **💾 Salva** per confermare

---

### 3. Riordina le sezioni
1. Torna indietro nelle impostazioni
2. Vai su **Riordina sezioni**

Qui puoi:

- spostare le sezioni **su o giù** usando le frecce
- organizzare l'ordine delle sezioni come preferisci

Quando hai finito:

👉 premi **💾 Salva**

---

### 4. Applicare le modifiche

Dopo aver salvato tutto:

1. **L'app si riavvierà**
2. Comparirà una nuova scheda chiamata **Homepage**

All'interno troverai **tutte le sezioni personalizzate** che hai scelto e ordinato.

---

## 💡 Suggerimenti

- Puoi combinare **sezioni di plugin diversi** nella stessa homepage
- Se modifichi l'ordine o le sezioni ricordati sempre di premere **💾 Salva**
- Se le modifiche non compaiono subito, **riavvia l'app**
